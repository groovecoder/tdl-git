Why GitHub?

  • Cloudy
  • Collaboration
  • Services
  • Open Source stuff

Cloudy

  • no server to maintain
  • backup and file sync
  • free web pages (source)
  • API

Collaboration

  • Issue tracking (extensions via API - e.g., Huboard)
  • Pull Requests & Code Reviews

Services

  • Notifications - RSS, Email, IRC, Jabber, more
  • Issue updates - GitHub (duh), Bugzilla, AgileBench, AgileZen, Basecamp, Campfire, FogBugz, Jira, Kanbanery, Trac, Trello, more
  • Continuous Integration - AppHarbor, Bamboo, Jenkins, Travis, TeamCity, more (auto-test pull requests)
  • Code quality - CodeClimate, etc.?
  • Docs - ReadTheDocs
  • Post-Receive web hooks

Open Source

  • Unlimited features for public/open-source repos
  • Everyone's doing it
